## Installation

To use these algorithms, you need to have [Node.js](https://nodejs.org/) installed. Follow these steps to set up the project:

1. Clone the repository:
   ```sh
   git clone ...
   ```
2. Navigate to the project directory:
   ```sh
   cd algorithm
   ```
3. Install the dependencies:
   ```sh
   npm install
   ```
